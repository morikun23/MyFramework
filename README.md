# MyFramework
自作フレームワーク用リポジトリ。<br>単に自分が使いやすくなるようにあらかじめ機能をここで整えておくことが目的である<br>

以下、運用ルール（随時、更新予定）

#### ◆ヘッダー項目一覧
- 【Unity】Unityプロジェクトのバージョン更新など<br>　※必ずバージョンも明記すること

- 【プロジェクト設定】プロジェクト設定(PlayerSettingsなど)に変更があった際に明記



#### ◆ブランチルール
- 【master】        -最新用ブランチ
- 【feature/...】   -機能開発用ブランチ


※個人用なのでこれくらいでいいか....。
